# smart-css

### CSS Break Points
- Mobile devices: 320-480 PXs
- iPad and tablets: 481-768 PXs
- Small screens (Laptops): 769-1024 PXs
- Large screens (Desktop): 1025-1200 PXs
- Extra large screens (TV): 1201 PXs and above
